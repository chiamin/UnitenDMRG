# Attribution and Technical Credits

### Overview

This repository contains a collection of computational tools and numerical implementations. The codebase is the result of a collaborative development process between the human developer and Artificial Intelligence (AI) assistants.

### Technical Inspiration & Prior Art

The algorithms, data structures, and logical workflows implemented in this project are inspired by established open-source libraries and numerical frameworks within the scientific computing community. These may include, but are not limited to:

* **ITensor** (itensor.org)
* **TeNPy** (Tensor Network Python)
* **Quimb**, **Block**, and other community-standard implementations.

While this project is an independent implementation, it acknowledges and follows the mathematical conventions and algorithmic patterns popularized by these pioneering works.

### AI Assistance Disclosure

A significant portion of the source code, logic scaffolding, and documentation in this repository was generated and structured with the assistance of **Large Language Models (LLMs)**.

* **Role of AI:** The AI provided initial implementations, code translation, and optimization suggestions based on broad patterns found in scientific software.
* **Human Oversight:** The developer performed the final architectural integration, debugging, and verification to ensure the physical and numerical consistency of the results.
